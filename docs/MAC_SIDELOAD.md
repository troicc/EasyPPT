# EasyPPT 本地安装

这是一个 PowerPoint Office.js Web Add-in，适合 macOS 版 PowerPoint 侧载测试。

## 运行本地服务

```sh
npm run build:native
npm run start
```

`build:native` 会编译素材库使用的 macOS 剪贴板助手。正常启动时若助手不存在或源码有更新，本地服务也会自动重新编译。

默认地址是：

```text
https://localhost:3000/src/taskpane.html
```

第一次运行会生成自签名证书：

```text
.certs/localhost.crt
```

如果 PowerPoint 任务窗格显示证书错误，把这个证书加入 macOS 钥匙串并设为始终信任，然后重启 PowerPoint。

## 侧载到 macOS PowerPoint

1. 打开 Finder。
2. 按 `Cmd` + `Shift` + `G`。
3. 前往：

```text
/Users/<你的用户名>/Library/Containers/com.microsoft.Powerpoint/Data/Documents/wef
```

如果 `wef` 文件夹不存在，就手动创建。

4. 把项目里的 `manifest.xml` 复制到这个 `wef` 文件夹。
5. 重启 PowerPoint。
6. 打开任意 PPT，进入 `开始` > `加载项`，选择 `EasyPPT`。

## 当前功能

- 复制和粘贴选中对象的位置、尺寸。
- 以第一个选中对象为参考做左对齐、居中、顶端对齐、垂直居中。
- 设置选中对象的水平或垂直间距。
- Figure 一键排版：不同尺寸图片等比例适配到统一画布，自动排列并在图片外添加 A–F 标签，同时生成整图标题和图注；可保存自定义期刊模板。
- 插入轻量 Markdown 文本块。
- 把 LaTeX 公式渲染为高清透明图片插入，兼容旧版 Mac PowerPoint。
- 经典配色库分为科研配色、氛围配色和传统配色，全部使用可视化配色卡浏览。
- 传统配色保留水绿、樱草、藤萝、石绿、秋波、颊红、田园、茶花、靛青、柠檬等颜色名称与 RGB。
- 可上传 PNG、JPG 或 WebP 论文图片，自动忽略白底、黑色坐标轴和透明区域，提取主要数据颜色。
- 可直接点选调色板中的任意颜色，应用到 PPT 选中对象的填充、轮廓线或文字；也可整套依次或循环分配。
- 先选择图类型，再在固定表格模板中填写数据；也可直接粘贴 CSV。支持折线、散点、分组柱状、重复值均值柱、面积、均值 ± 误差、箱线和热图，可插入高清图或下载 SVG。
- 数据标签可按图型显示单点数值、名称＋数值或统计摘要，并可设置小数位和字号；重复值柱支持 mean、SD、n 与每个重复点，箱线图支持中位数、四分位数、n 与原始点。
- 散点图可为每个 Y 系列计算线性回归，并分别控制回归线、公式与 R² 的显示。
- 绘图模块可直接载入经典配色中的当前色板，也可逐个修改系列颜色；支持 EasyPPT、GraphPad（按 R/ggprism 的 theme_prism 视觉规则实现）、Classic Journal、Python/Matplotlib、R/ggplot2 和 Minimal 风格，并可设置默认风格。
- 支持小提琴图＋原始点，以及左右轴量纲独立的柱形＋折线叠加图；高级外观可设置字体字号、标签位置、线宽、点大小、透明度、横轴旋转、图例位置、背景与坐标轴颜色。
- 图表配色包含经典配色模块中的全部色板。先点击系列颜色槽位，再点击完整色库中的颜色进行分配；只有“自定义颜色…”按钮会打开系统取色器。
- 双 Y 轴图默认将右轴刻度、右轴标题和图例分区排布，并可设置图例水平/垂直偏移及右轴文字间距。切换图型时会显示“本图重点”并高亮相关控件。
- 双轴柱值与折线值标签接近时会自动错位，并添加背景描边保证可读性。系列颜色区只显示当前数据实际使用的系列，并会随插件侧栏宽度自动换行。
- 坐标轴提供自动规范定标和手动设置。自动模式会向外取整覆盖全部数据，使用 1、2、2.5、5 × 10ⁿ 主刻度；手动模式可设置最小值、最大值、主刻度间隔及次刻度数。双轴图左右轴独立设置。
- 配色控件随图型变化：单一元素图只显示实际需要的颜色；多系列图使用配色方案，高级折叠区可逐系列覆盖。
- EasyPPT、GraphPad、Classic Journal、Matplotlib、ggplot2 与 Minimal 风格分别具有不同轴框、刻度、网格、字体、绘图区和默认色板，不再只是更换背景色。
- 数据标签默认使用自动高对比颜色，并可切换为手动颜色或加粗。热图默认由低值色向高值色连续渐变，中值颜色为可选覆盖。
- “我的数据集”可保存当前真实表格、图型及轴标题；同名保存会更新，之后可打开继续修改或删除。
- 根据 DOI 查询 Crossref 元数据，生成完整引用或“期刊缩写、年份、卷(期)、页码”的短引用。
- 引用支持设置字体、字号、颜色、位置和宽度；短引用插入时自动设置期刊名与卷期斜体、年份粗体。
- 每个功能模块标题右侧提供上移和下移按钮，排列顺序会自动保存在本机。
- 本地素材库通过 PowerPoint 原生复制/粘贴保存和插入对象组合，支持文本框、照片、组合、图表和特殊形状，并保持可编辑。
- 通过兼容 OpenAI Chat Completions 的接口润色选中文本。
- 导出当前页为 PNG、JPEG 或 WebP，并将当前页、已选页面或全部页面批量合并导出为多页 PDF。

## 已知边界

- 素材库首次使用时，macOS 会询问是否允许本地服务自动化控制 Microsoft PowerPoint；需要选择允许。
- 原生素材数据保存在项目目录的 `.native-assets/` 中。仅导出 JSON 索引不能迁移原生素材文件。
- AI 请求是否可用取决于接口是否允许 Office 任务窗格跨域调用；如果被 CORS 拦截，需要接一个本地代理服务。
- EasyPlot 插入 PowerPoint 后是高清图片；下载的 SVG 可继续在矢量编辑器中修改。
