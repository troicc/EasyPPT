# EasyPPT

一个面向 macOS PowerPoint 的科研 PPT 效率助手。

## 快速开始

```sh
npm run start
```

然后按 [docs/MAC_SIDELOAD.md](./docs/MAC_SIDELOAD.md) 把 `manifest.xml` 侧载到 PowerPoint。

## 功能模块

- 位置尺寸复制粘贴
- 对齐和间距增强
- Figure 一键排版：不同尺寸图片等比例适配、统一画布与间距、图片外 A–F 标签、整图标题和图注，并可保存期刊模板
- Markdown 插入
- LaTeX 高清公式插入
- 经典配色库：科研配色、氛围配色、传统配色三类可视化方案；传统颜色保留水绿、藤萝、秋波等名称
- 固定表格模板与 CSV 双输入模式，支持折线、散点、分组柱状、重复值均值柱、面积、均值 ± 误差、箱线和热图
- 图标题、坐标轴、配色、网格和图例设置；数据标签支持单点值、名称＋数值、统计摘要、全部信息、小数位与字号
- 散点图支持多系列线性回归，可独立显示回归线、回归公式和 R²
- 新增小提琴图与双 Y 轴柱线叠加图；绘图可直接使用“经典配色”模块当前选中的科研色板
- 提供 EasyPPT、GraphPad（按 R/ggprism 的 theme_prism 视觉规则实现）、Classic Journal、Python/Matplotlib、R/ggplot2 与 Minimal 风格，可设置默认风格，并可微调字体、标签位置、线宽、点大小、坐标轴、背景和系列颜色
- 图表配色下拉与点选色库包含经典配色模块的全部科研、氛围和传统色板；系列颜色采用“选择槽位 → 点选颜色”的方式
- 双轴图为右轴刻度、右轴标题和图例预留独立空间，并支持图例 X/Y 偏移与右轴文字间距；各图型动态提示并高亮最重要的设置
- 双轴图数据标签带自动碰撞检测：柱值与折线值接近时会自动上下、左右错位；系列色块在窄侧栏自动换行且不会横向溢出
- 坐标轴支持规范自动定标与手动范围：主刻度采用 1、2、2.5、5 × 10ⁿ，支持主/次刻度、数值 X 轴以及双轴独立范围
- 配色按图型精简：重复值柱只显示柱色与原始点色，双轴图只显示柱色与折线色，热图显示低/中/高值色；多系列图才显示配色方案与高级逐系列覆盖
- 五套风格现在分别控制轴框、主次刻度、绘图区背景、网格、字体字重、线宽、点大小和默认色板
- 数据标签默认自动选择与画布和数据系列均有高对比度的黑/白颜色，也可切换为自定义颜色并设置加粗
- 热图默认只设置低值色与高值色并自动连续渐变；需要发散色板时可开启自定义中值颜色
- 真实输入数据可命名保存到“我的数据集”，支持重新打开、修改、同名更新和删除
- DOI 元数据查询、完整/短引用、富文本格式插入和本地引用库
- 模块顺序自定义并自动保存
- 本地素材库（通过 macOS 原生剪贴板助手保存文本框、图片、组合、图表和特殊形状，恢复后保持 PowerPoint 可编辑）
- AI 文本润色
- 智能动画：按标题、正文、图表、图片和强调元素自动安排动画，并参考上下页连续性选择淡入、擦除、缩放与 Morph 过渡
- 当前页 PNG、JPEG、WebP 导出，以及当前页、已选页面或全部页面的多页 PDF 合并导出；支持标准、高清、超高清和自定义清晰度

## 技术说明

macOS 上不能直接使用 Windows VSTO/COM 插件架构，所以主体使用 PowerPoint Office.js Web Add-in。素材库额外使用本地 Swift 剪贴板助手和 PowerPoint AppleScript 接口，以保留 Office 原生对象格式。
