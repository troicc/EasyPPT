## 小红书正文

导师以为我最近做 PPT 速度变快，是因为熟能生巧。
其实是因为我把最折磨人的几步，塞进了 PowerPoint 侧边栏。🙂

EasyPPT 是一个面向 Mac PowerPoint 的科研效率助手。第一版先解决 5 件高频小事：

📊 CSV 数据直接生成科研图
🧮 Markdown / LaTeX 插入高清公式
📚 DOI 查询并生成长、短引用
🎨 一键插入科研章节过渡页
🖼️ PNG、WebP、高清图和多页 PDF 导出

它不是替你编内容，也不是套一个花哨模板。
它做的事情更朴素：把那些每天重复、每次都烦、又不得不做的步骤缩短。

以前：数据 → 作图软件 → 导出 → PPT → 调格式。
现在：数据 → PPT。

以前：查 DOI → 复制作者 → 改标点 → 调字号。
现在：贴 DOI → 插入。

科研已经够难了，PPT 就别再给自己加实验组了。

想试用或想看后续功能，评论区留「EasyPPT」👇
